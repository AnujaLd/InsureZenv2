## 📡 ALL 12 API ENDPOINTS WORKING

| # | Endpoint | Method | Purpose |
|---|----------|--------|---------|
| 1 | /auth/register | POST | Register new user |
| 2 | /auth/login | POST | Login and get JWT |
| 3 | /claims/ingest | POST | Submit claim |
| 4 | /claims/{id} | GET | Get claim details |
| 5 | /claims/maker/list | GET | Claims for maker |
| 6 | /claims/checker/list | GET | Claims for checker |
| 7 | /claims/{id}/lock/maker | POST | Lock as maker |
| 8 | /claims/{id}/lock/checker | POST | Lock as checker |
| 9 | /claims/{id}/unlock | POST | Release lock |
| 10 | /claims/{id}/review/maker | POST | Maker review |
| 11 | /claims/{id}/review/checker | POST | Checker review |
| 12 | /claims/{id}/forward-to-insurer | POST | Forward claim |

**Every endpoint tested and working**
